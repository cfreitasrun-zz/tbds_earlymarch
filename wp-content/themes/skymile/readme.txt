== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Skymile in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. For changing settings and customize theme: Navigate to Appearance > Customize in admin panel.
5. For theme option: Navigate to Appearance > Customize > Theme Option.
6. Customize Header section : Navigate to Appearance > Customize > Theme Option > Header.
7. Customize Slider : Navigate to Appearance > Customize > Theme Option > Slider setting.
8. Customize Home features : Navigate to Appearance > Customize > Theme Option > Home Feature Setting.
3. Customize Home Tagline : Navigate to Appearance > Customize > Theme Option > Home Page Tag line.
10. Adding social icons : Navigate to Appearance > Customize > Theme Option > Social Icons.
11. Customize footer details : Navigate to Appearance > Customize > Theme Option > Footer Settings.
12. Customize footer details : Navigate to Appearance > Customize > Theme Option > Footer Settings.
13. Customize site details : Navigate to Appearance > Customize > Site Identity.
14. Customize Menus : Navigate to Appearance > Customize > Menus.
15. Customize Widgets : Navigate to Appearance > Customize > Widgets.
